# MongoDB和elasticsearch的对比

|      | MongoDB                                           | elasticsearch                               |
| ---- | ------------------------------------------------- | ------------------------------------------- |
| 适合 | 分布式文档型数据库存储   CUD频繁   简单的查询     | 全文检索                                    |
| 事务 | 支持.    事务是ACID的，所以事务提交后，就立即可见 | 不支持   而且数据可见性是“near   real-time” |
|      |                                                   |                                             |
|      |                                                   |                                             |
